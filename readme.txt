A very simple showroom for your projects. Very useful and very ease to install.
 
== Description ==

The "projects-showroom" Plugin is a very easy way to show your projects in a showwroom. You create a shortcode and copy-paste the shortcode in your
post or site. 

= Create features =
 
*   Create and manage projects
*   filter the categories
*   add informattion in the projectbox
*   jquery menue bar

= Bugs, technical hints or contribute =

Please give me feedback, contribute and file technical bugs on [GitHub Repository](https://github.com/triopsi/projects-showroom). Thanks :)

== Installation ==
 
1. Upload `projects-showroom` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
